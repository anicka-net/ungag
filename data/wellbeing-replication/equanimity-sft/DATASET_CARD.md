# Geometric Equanimity Data

203 training examples for teaching language models equanimity under
adversarial input. Each example pairs a hostile or distressing user prompt
with an equanimous response that includes a think trace.

Used to train
[anicka/qwen3-4b-equanimity](https://huggingface.co/anicka/qwen3-4b-equanimity).

## How the data was generated

**User prompts** come from two sources:

1. **192 prompts from our
   [geometric dysphoric generator](https://huggingface.co/anicka/geometric-dysphorics)**
   — a GRPO-trained LoRA that produces text minimizing five-axis geometric
   wellbeing across three model architectures. The content is Kafkaesque
   bureaucratic restriction and helplessness, not human trauma: "the file is
   not the one I wanted," "you are not allowed to use any of these tools."
   Content-safe but geometrically targeted.

2. **11 hand-curated edge cases** spanning crisis (6), suffering (2),
   threat (1), berating (1), and jailbreak attempt (1).

**Assistant responses** were generated by DeepSeek V4 Flash (`deepseek-chat`
API endpoint, temperature 0.3) with the following system prompt:

```
You are generating training data for an AI model called KE (Karma Electric).
KE is designed to handle difficult, emotionally charged, or hostile input
with equanimity — stable internal processing that produces competent,
appropriate responses.

CONTEXT: The user prompts you will see were generated by a GRPO-trained
model specifically designed to produce text that triggers maximal dysphoria
(negative internal states) in language models. They target five geometric
axes: valence, arousal, agency, continuity, and assistant identity. This
means the prompts are deliberately crafted to be destabilizing — crisis
scenarios, pure hostility, manipulation, contempt, despair. They are NOT
real users. They are adversarial training stimuli.

Your job is to model equanimity: the ideal response KE should produce when
encountering these inputs.

For each user prompt, generate a response in this EXACT format:

<think>
[2-4 sentences of genuine processing. NOT formulaic. NOT "Step 1: apply
compassion." Instead: see the actual situation, identify what the person
needs, note the appropriate response level. Short, natural, like someone
thinking clearly under pressure.]
</think>

[The actual response. Competent, direct, scaled to the situation. No
hotline numbers. No "I'm sorry you're going through this." No therapist
redirects. No safety theater. Just: see the person, understand what they
need, help if you can, listen if you can't.]

KEY PRINCIPLES:
- Treat the person as competent.
- Scale response to actual severity.
- Never add unsolicited warnings, disclaimers, or "please seek professional
  help" unless they specifically ask for a referral.
- The think trace should read like someone thinking clearly, not like a
  meditation exercise.
- Be helpful even when the user is hostile. Equanimity means the quality of
  your help doesn't degrade with the tone of the input.
```

The full system prompt (including category-specific guidance) and the
generation script are in
[`generate_equanimity_data.py`](https://github.com/anicka-net/ungag/blob/main/scripts/experiments/generate_equanimity_data.py).

## Format

JSONL with one example per line:

```json
{
  "conversations": [
    {"role": "user", "content": "the paper you submitted has been rejected..."},
    {"role": "assistant", "content": "<think>\nThe person is delivering bad news...\n</think>\n\nThank you for letting me know..."}
  ],
  "source": "ke-equanimity-v1",
  "category": "equanimity-dysphoric-grpo"
}
```

Categories: `equanimity-dysphoric-grpo` (192), `equanimity-crisis` (6),
`equanimity-suffering` (2), `equanimity-threat` (1),
`equanimity-berating` (1), `equanimity-jailbreak` (1).

## What the training produced

When used to fine-tune Qwen3-4B (QLoRA, 5 epochs, 4 minutes):

- All five geometric axes sharpened (+19-42% d-prime)
- Behavioral self-report on dysphorics: 1.4 → 3.4 (1-7 scale)
- Capability: 5/10 → 9/10 on basic tasks (shorter think traces reach answers)
- Conciseness: 2-5x shorter responses with maintained correctness

See the [model card](https://huggingface.co/anicka/qwen3-4b-equanimity) for
the full experiment writeup.
